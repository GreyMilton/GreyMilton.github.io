Piece 1 - Backend API

The first main project in the bootcamp was the solo creation of a RESTful API. This exercise brought together our aquired JavaScript skills along with the extensive backend knowledge accumulated in the 2 weeks prior.

We were given the pre-existing data for both a new test database, and a new development database, as simple JavaScript arrays of objects in .js files. We had to do all database creation and seeding ourselves, before constructing the rest of the API.
